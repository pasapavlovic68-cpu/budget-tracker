# Budget Tracker

## Instructions for Mobile Use

1. **Accessing the App**: You can access the Budget Tracker app via your mobile browser by visiting [your app's URL].
2. **Saving to Home Screen**: To save the app to your home screen, follow these steps:
   - For iOS:
     - Open your browser (Safari) and navigate to the app.
     - Tap on the "Share" icon at the bottom of the screen.
     - Select "Add to Home Screen".
   - For Android:
     - Open your browser (Chrome) and navigate to the app.
     - Tap on the three dots menu icon at the top-right corner.
     - Select "Add to Home screen".
3. **Using the App**: Once added, you can open the app just like any other app on your device. Navigation and functionality are optimized for mobile use.

## PWA Installation

1. **Installing the PWA**: When accessing the app on a supported browser (like Chrome or Firefox), a prompt will appear asking if you want to install the app. Confirm the prompt to add the Progressive Web App to your device.

2. **Accessing Offline Features**: Once installed, the PWA allows you to access certain features even when you are offline, enhancing the user experience.

3. **Updating the App**: The PWA will automatically update to the latest version whenever an internet connection is available, ensuring you always have the latest features and bug fixes.
